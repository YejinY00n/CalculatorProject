## 1. 프로젝트 소개
> 사용자로부터 수를 입력받아 사칙연산을 수행하는 `Java` 기반의 **계산기** 프로그램

<br>

### 2. 기능
✅ 사칙연산 (덧셈, 뺄셈, 곱셈, 나눗셈)
✅ 현재 연산 결과값 보다 큰 이전의 연산 결과값들 출력
✅ 예외 처리 (사용자 입력, 나눗셈 etc.)
✅ 실행 종료
